A useful test is: Given input x, would a human expert be able to confidently predict y?</p><p><strong>Rationale for large data</strong>: if we have a <strong>low bias</strong> algorithm (many features or hidden units making a very complex function), then the larger the training set we use, the less we will have overfitting (and the more accurate the algorithm will be on the test set).</p><h1>Quiz instructions</h1><p>When the quiz instructions tell you to enter a value to "two decimal digits", what it really means is "two significant digits".
